# Weka Data Mining en GitHub Codespaces

## Arranque del entorno

1. Abre este repositorio en GitHub.
2. Pulsa `Code`.
3. Selecciona `Codespaces`.
4. Pulsa `Create codespace on main`.
5. Cuando se abra VS Code Web, abre una terminal.

## Iniciar el escritorio web

Ejecuta:

```bash
./scripts/start-desktop.sh
```

Abre la pestaña PORTS y entra en:

Weka Desktop noVNC

## Iniciar Weka

Abre una segunda terminal y ejecuta:

```bash
./scripts/start-weka.sh
```
